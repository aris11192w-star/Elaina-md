T5# Elaina-md

## Instalasi awal

1. Update package

```bash
apt update && apt upgrade
```

2. Install nodejs

```bash
apt install nodejs -y
```

3. Install ffmpeg (ini diperlukan untuk mengkonversi ke format webp terutama pada "stiker")

```bash
apt install ffmpeg -y
```

4. Install git

```bash
apt install git
```

5. Cloning repo

```bash
https://github.com/aris11192w-star/Elaina-md.git
```

6. unzip file

```bash
cd Elaina-md && unzip Elaina-R-md && npm install && npm start 
```
